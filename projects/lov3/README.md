# LOV3 — 프로젝트 미디어

옵시디언 노트는 `03. 프로젝트/LOV3/`. **무거운 파일은 여기** 두고, 스틸은 볼트 `LOV3/data/` 에도 복사해 임베드.

## `media/`

| 경로 | 내용 |
|------|------|
| `media/official/` | 공식 뮤비 mp4 (다운로드) |
| `media/recordings/` | 화면 녹화 mp4 (`녹음 …`) |
| `media/stills/mv/` | 뮤비에서 뽑은 선별 스틸 7장 |
| `media/stills/recordings/` | 녹화 분석 스틸 + `_meta.json` |

코드·웹 빌드가 생기면 같은 폴더에 `src/` 등을 두면 됨.
