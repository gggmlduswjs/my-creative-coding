# my-creative-coding

크리에이티브 코딩 실험 레포.

## 폴더

| 경로 | 역할 |
|------|------|
| `projects/` | 프로젝트별 코드·미디어 (예: `projects/lov3/` → [README](projects/lov3/README.md)) |
| `assets/` | 공용 에셋 (가벼운 것만; 큰 mp4는 프로젝트 `media/` 권장) |
| `core/`, `effects/` | 공용 모듈·이펙트 |

## LOV3

미디어는 **`projects/lov3/media/`** 아래 `official` · `recordings` · `stills` 로 구분.
